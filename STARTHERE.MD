# Start Here (TL;DR)

## Minimum Requirements
- `git`
- `claude` (Claude Code CLI)

## 1) Clone to `~/.dotfiles`
```bash
git clone git@github.com:cs3b/dotfiles-setup-env.git .dotfiles
```

## 2) Start the Agent in This Folder
```bash
cd ~/.dotfiles
claude
```

That is the minimum setup to start working.
